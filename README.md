# 🍽️ RecipeVerse

RecipeVerse is a modern and responsive recipe website built using **HTML**, **CSS**, and **JavaScript**. It provides users with an easy way to discover delicious recipes, view ingredients, and follow step-by-step cooking instructions through a clean and intuitive interface.

# ✨ Features

* 🥘 Browse a variety of recipes
* 🔍 Search for recipes
* 📖 View detailed recipe information
* 📝 Ingredient and cooking instructions
* 📱 Fully responsive design
* 🎨 Clean and modern user interface
* ⚡ Fast and lightweight

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
